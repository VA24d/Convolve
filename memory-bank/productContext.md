# Product Context

## Why This Exists
Rural citizens struggle to navigate complex welfare scheme eligibility requirements. Yojana-Drishti reduces this friction by using AI to extract eligibility signals and match schemes with explainable evidence.

## User Experience Goals
- Simple input of state/caste/land/assets/intent.
- Optional vision analysis for signals from photos.
- Clear explanations for why a scheme matches.
- Memory recall of similar prior cases.

## Primary Interfaces
- Streamlit demo UI for desktop exploration.
- Expo mobile app for field usage, now running orchestration on-device.
