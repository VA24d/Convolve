# Tech Context

## Stack
- Python (FastAPI, LangChain, Qdrant client) for backend services.
- Qdrant Cloud for vector storage and filtered search.
- Expo (React Native) mobile app.
- LangChain.js (`@langchain/openai`) for mobile embeddings.

## Configuration
- `.env` for backend keys.
- `mobile/config.ts` for mobile OpenAI/Qdrant credentials.

## Tooling
- Use `pnpm` for JavaScript dependencies in the repo.
